🎇 Diwali Sales Analysis
📌 Overview

This project performs Exploratory Data Analysis (EDA) on Diwali sales data using Python. The goal is to analyze customer purchasing behavior and identify key business insights that can help improve sales strategies.

🎯 Objectives
Analyze customer demographics (Gender, Age, Marital Status)
Identify top-performing states and regions
Find most popular product categories
Understand spending patterns across occupations
🛠️ Tech Stack
Python
NumPy
Pandas
Matplotlib
Seaborn
Jupyter Notebook
📂 Dataset Features

The dataset includes:

User_ID
Gender
Age Group
State
Marital Status
Occupation
Product Category
Orders
Amount
🧹 Data Cleaning
Removed irrelevant columns (Status, unnamed1)
Handled missing values using dropna()
Converted Amount column to integer
Checked and fixed inconsistencies
